WELCOME TO HTML

HTML stands for HYPER TEXT MARKUP LANGUAGE. HTML is the standard markup language for documents designed to be displayed in a web browser.
HTML is easy to learn and quite powerful.

Due to its evolution nature, HTML undergoes certain versions, namely : -
* HTML 1.0
* HTML 2.0
* HTML 3.0
* HTML 3.2
* HTML 4.0
* XHTML 1.0
* HTML 5.0
